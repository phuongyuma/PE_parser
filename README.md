# PE_parser
 
### List information of file (c++)
- PointerToEntryPoint
- CheckSum
- ImageBase
- FileAlignment
- SizeOfImage
- Infomation of each section:
    - Name
    - Characteristics
    - RawAddress
    - RawSize
    - VirtualAddress
    - VirtualSize
- Functions the file import and exports (incomplete)

Command to run: 
`./PE_Parser.exe <file path>`
