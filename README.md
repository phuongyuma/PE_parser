# PE_parser
 
### List information of file (c++)
- PointerToEntryPoint
- CheckSum
- ImageBase
- FileAlignment
- SizeOfImage
- Infomation of each section:
    - Name
    - Characteristics
    - RawAddress
    - RawSize
    - VirtualAddress
    - VirtualSize
- Functions the file import and exports (incomplete)

Result when run with notepad.exe (64bit)
